# Hadee의 Todo
## Objective
Link: https://hadee-todo.netlify.app/

Vue 3를 공부하면서 저만의 Todo를 만들었습니다.

**1주일간의 진행**을 그래프로 볼 수 있게하여 **차별화**를 확보하였습니다.
